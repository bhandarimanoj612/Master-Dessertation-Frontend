"# Master-Dissertation-Frontend

## Overview

The frontend component of the SaaS Repair Service Management System, built with React and TypeScript. Provides a modern, responsive user interface for customers, technicians, and administrators.

## Features

- Responsive dashboard for different user roles
- Appointment booking and management
- Real-time repair status updates
- Inventory tracking interface
- Billing and invoice viewing
- Admin panel for system management
- Interactive maps for location services

## Tech Stack

- **Framework**: React 18
- **Language**: TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS, Material-UI
- **Routing**: React Router
- **HTTP Client**: Axios
- **Charts**: Recharts
- **Maps**: Leaflet
- **Animations**: Framer Motion

## Prerequisites

- Node.js 18+
- npm or yarn

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repo.git
   cd your-repo/Frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Application

1. Start the development server:
   ```bash
   npm run dev
   ```

2. Build for production:
   ```bash
   npm run build
   ```

3. Preview production build:
   ```bash
   npm run preview
   ```

## Project Structure

```
src/
├── pages/          # Page components
├── global/         # Shared components and utilities
├── routes/         # Routing configuration
└── assets/         # Static assets
```
